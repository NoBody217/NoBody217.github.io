T1: L1-094 剪切粘贴

题意

  • 给定字符串,多次截取指定区间的子串,拼接到目标子串的前/后,若找不到则拼在末尾.
  • 昨天模考的“心态粉碎机”,今天完美复仇.

思路

  • 遵循“先提取 -> 再抹除 -> 再寻找 -> 最后拼接”的流水线.
  • s.substr() 提取子串,s.erase() 删去原位置,在干净的原串中用 s.find() 找目标位置,最后再次用 substr 分段拼接.
  • 注意算好位置和长度再写,别乱写然后改,浪费时间

T3: L2-043 龙龙送外卖

题意

  • 给定一棵树,外卖站为根.每次新增一个送餐点,外卖员从根出发送完所有已知点,不用回去.
  • 求每次新增订单后的最小总行走距离.

思路

  • 树上贪心 (思维题)
  • 如果必须回外卖站,总距离 = 2 * 配送树的总边数.
  • 既然不需要回去,贪心策略就是：把离外卖站最远的那一单留在最后送！
  • ans = 2 * 总边数 - 当前订单中的最大深度.
  • 维护策略：
    • 深度：记忆化搜索,用 vis[x] 充当记忆化数组,vis[x] = dfs(fa[x]) + 1
    • 边数：每次来新订单,顺着 fa 向上爬,爬到曾经访问过的地方为止,爬了几步就新增了几条边.全局均摊 O(N)O(N)O(N).
  • 不要用cerr,会超时

T4: PAT-1146 Topological Order

题意

  • 给定一张有向图,判断给出的若干个序列是否为合法的拓扑排序序列.

思路

  • 拓扑排序验证
  • 建图时记录每个节点的初始入度 deg.
  • 对于每个查询序列：
    • 拷贝一份入度数组 ndeg = deg.
    • 遍历序列中的每个点 x：
      • 如果 ndeg[x] != 0,说明它还有前置任务没完成,绝对不是拓扑序！
      • 如果 ndeg[x] == 0,说明合法,模拟将它删去,把它所有邻居 v 的入度减一 (ndeg[v]--).
  • 拓扑排序的求法
    • 用一个队列,每次把度为0的点扔进去
    • 拿出队首,更新入度数组
