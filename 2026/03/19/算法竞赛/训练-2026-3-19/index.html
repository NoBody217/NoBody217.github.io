T2: L2-023 图着色问题

题意

  • 给定一张无向图和每种颜色的编号。验证所给方案是否满足：
    1. 使用的颜色总数恰好等于 KKK 种。
    2. 任意一条边的两个端点颜色不同。

思路

  • 别读错题,注意是恰好k个
  • 图染色合法性校验。
  • 使用 std::set 收集当前方案中所有的颜色编号
  • 遍历所有图中的点 uuu 及其邻居 vvv，若发现 color[u] == color[v]，则立即判定不合法

T3: PAT甲级 1122 Hamiltonian Cycle (哈密顿环)

题意

  • 判断给定的顶点序列是否构成了一个哈密顿环
  • 哈密顿环定义：从某个顶点出发，经过图上每一个顶点恰好一次，最后回到起始顶点的一条路径。

思路

  • 切忌边读边校验，先存后处理
  • 合法的哈密顿序列必须同时满足以下四重校验：
    1. 长度校验：包含了 NNN 个顶点并回到起点，因此序列长度必须恰好为 N+1N + 1N+1。
    2. 首尾校验：path.front() == path.back()。
    3. 点集校验：将序列放入 set 去重，size必须恰好等于 NNN。
    4. 连通性校验：遍历序列中的每对相邻元素 path[i]path[i]path[i] 和 path[i+1]path[i+1]path[i+1]，在邻接矩阵中检查它们之间是否存在有效边

T4: PAT甲级 1126 Eulerian Path (欧拉路径)

题意

  • 给定一个无向图，判定它是欧拉图（Eulerian）、半欧拉图（Semi-Eulerian）还是非欧拉图。

思路

  • 根据离散数学定理，判定欧拉图的充分必要条件：
    1. 连通性：拿并查集做
    2. 顶点个数–拿degre判
      • 若 odd == 0，则是欧拉图（可一笔画并回到起点）。
      • 若 odd == 2，则是半欧拉图（可一笔画但起点终点不同）。
      • 否则，为非欧拉图。
