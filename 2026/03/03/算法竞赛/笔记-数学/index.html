数学

一、 质数与约数 (Prime Numbers & Divisors)

1. 质数基础性质

  • 奇偶性：n>3n > 3n>3 时，nnn 和 n+1n+1n+1 总有一个是偶数（不是质数）。
  • 无限性：质数有无穷多个。
  • 分布稀疏性：
    • 随着 nnn 的增大，质数越来越稀疏。
    • NNN 以内的质数个数 π(N)\pi(N)π(N) 趋近于 Nln⁡(N)\frac{N}{\ln(N)}ln(N)N​。
    • 伯特兰-切比雪夫定理：aaa 到 2a2a2a 之间至少存在一个质数（a>1a>1a>1）。
  • 任意大间隔：存在任意长的一段连续自然数，其中所有数都是合数。
    • 构造证明：令 a=(n+1)!a = (n+1)!a=(n+1)!，则 a+2,a+3,…,a+(n+1)a+2, a+3, \dots, a+(n+1)a+2,a+3,…,a+(n+1) 连续 nnn 个数均为合数。

2. 著名猜想

  • 孪生素数猜想：存在无穷多对 (p,p+2)(p, p+2)(p,p+2) 均为质数的情况（如 3和5, 11和13, 1e9+7和1e9+9）。
  • 哥德巴赫猜想：任一大于 2 的偶数都可以写成两个质数之和。

4. 算术基本定理 (唯一分解定理)

任何大于 1 的正整数都能唯一分解成有限个质数的乘积。

  • 标准分解式：N=p1c1p2c2…pkckN = p_1^{c_1} p_2^{c_2} \dots p_k^{c_k}N=p1c1​​p2c2​​…pkck​​
  • 推论 1（约数个数）：NNN 的正约数个数为 (c1+1)(c2+1)…(ck+1)(c_1+1)(c_2+1)\dots(c_k+1)(c1​+1)(c2​+1)…(ck​+1)。
  • 推论 2（约数之和）：NNN 的正约数之和为 (p10+⋯+p1c1)…(pk0+⋯+pkck)(p_1^0 + \dots + p_1^{c_1}) \dots (p_k^0 + \dots + p_k^{c_k})(p10​+⋯+p1c1​​)…(pk0​+⋯+pkck​​)。

二、 最大公约数与欧拉函数 (GCD & Euler’s Totient)

1. GCD 与 LCM

  • 辗转相除法 (欧几里得算法)：gcd⁡(a,b)=gcd⁡(b,a mod b)\gcd(a,b) = \gcd(b, a \bmod b)gcd(a,b)=gcd(b,amodb)。
  • 最小公倍数：lcm(a,b)=a×bgcd⁡(a,b)\text{lcm}(a,b) = \frac{a \times b}{\gcd(a,b)}lcm(a,b)=gcd(a,b)a×b​。

2. 欧拉函数 φ(n)\varphi(n)φ(n)

  • 定义：1∼n1 \sim n1∼n 中与 nnn 互质的数的个数。
  • 计算公式：φ(n)=n×(1−1p1)×⋯×(1−1pk)\varphi(n) = n \times (1 - \frac{1}{p_1}) \times \dots \times (1 - \frac{1}{p_k})φ(n)=n×(1−p1​1​)×⋯×(1−pk​1​)，其中 pip_ipi​ 为 nnn 的质因子。
  • 积性函数性质：若 gcd⁡(a,b)=1\gcd(a,b)=1gcd(a,b)=1，则 φ(a×b)=φ(a)×φ(b)\varphi(a \times b) = \varphi(a) \times \varphi(b)φ(a×b)=φ(a)×φ(b)。

三、 同余与模算术 (Modular Arithmetic)

1. 同余基础

  • 定义：aaa 和 bbb 除以 mmm 的余数相等，记作 a≡b(modm)a \equiv b \pmod ma≡b(modm)。
  • 剩余系：模 mmm 的所有同余类构成完全剩余系：0‾,1‾,…,m−1‾\overline{0}, \overline{1}, \dots, \overline{m-1}0,1,…,m−1​。

2. 核心定理

  • 费马小定理：若 ppp 是质数，且 gcd⁡(a,p)=1\gcd(a, p)=1gcd(a,p)=1，则 ap−1≡1(modp)a^{p-1} \equiv 1 \pmod pap−1≡1(modp)。
    • 应用：求逆元，aaa 模 ppp 的逆元为 ap−2a^{p-2}ap−2。
  • 欧拉定理：若 gcd⁡(a,n)=1\gcd(a, n)=1gcd(a,n)=1，则 aφ(n)≡1(modn)a^{\varphi(n)} \equiv 1 \pmod naφ(n)≡1(modn)。
    • 应用：欧拉降幂公式，当 bbb 很大时，计算 ab(modn)≡ab mod φ(n)(modn)a^b \pmod n \equiv a^{b \bmod \varphi(n)} \pmod nab(modn)≡abmodφ(n)(modn)。

3. 裴蜀定理与扩展欧几里得 (ExGCD)

  • 裴蜀定理：对于任意整数 a,ba, ba,b，必然存在整数 x,yx, yx,y 满足 ax+by=gcd⁡(a,b)ax + by = \gcd(a,b)ax+by=gcd(a,b)。
  • ExGCD 算法：求解 ax+by=gcd⁡(a,b)ax + by = \gcd(a,b)ax+by=gcd(a,b) 的一组特解。

4. 二元一次不定方程 ax+by=cax + by = cax+by=c

  • 有解条件：当且仅当 gcd⁡(a,b)∣c\gcd(a,b) \mid cgcd(a,b)∣c 时有解。
  • 通解公式（已知特解 x0,y0x_0, y_0x0​,y0​）：
    • x=x0+bgcd⁡(a,b)×tx = x_0 + \frac{b}{\gcd(a,b)} \times tx=x0​+gcd(a,b)b​×t
    • y=y0−agcd⁡(a,b)×ty = y_0 - \frac{a}{\gcd(a,b)} \times ty=y0​−gcd(a,b)a​×t （ttt 为任意整数）
  • 取模归一化:利用同余性质获得最小非负解x′x'x′
    • x′=(x mod bgcd(a,b)+bgcd(a,b)) mod bgcd(a,b)x'=(x \text{ mod }\frac{b}{gcd(a,b)}+\frac{b}{gcd(a,b)})\text{ mod }\frac{b}{gcd(a,b)}x′=(x mod gcd(a,b)b​+gcd(a,b)b​) mod gcd(a,b)b​
    • 同时可以反解对应的y′y'y′

模运算中的除法等价于乘以逆元：ab(modm)≡a×b−1(modm)\frac{a}{b} \pmod m \equiv a \times b^{-1} \pmod mba​(modm)≡a×b−1(modm)。求 bbb 逆元的方法：

四、 组合数学 (Combinatorics)

1. 排列与组合

  • 排列：Anm=n!(n−m)!A_n^m = \frac{n!}{(n-m)!}Anm​=(n−m)!n!​
  • 组合：Cnm=(nm)=n!m!(n−m)!C_n^m = \binom{n}{m} = \frac{n!}{m!(n-m)!}Cnm​=(mn​)=m!(n−m)!n!​
  • 核心性质：
    • 对称性：Cnm=Cnn−mC_n^m = C_n^{n-m}Cnm​=Cnn−m​
    • 帕斯卡/杨辉三角递推：Cnm=Cn−1m+Cn−1m−1C_n^m = C_{n-1}^m + C_{n-1}^{m-1}Cnm​=Cn−1m​+Cn−1m−1​
    • 二项式系数和：∑i=0nCni=2n\sum_{i=0}^{n} C_n^i = 2^n∑i=0n​Cni​=2n（等价于 nnn 个元素选与不选的子集总数）。

2. 二项式定理

  • (a+b)n=∑i=0nCnian−ibi(a+b)^n = \sum_{i=0}^{n} C_n^i a^{n-i} b^i(a+b)n=∑i=0n​Cni​an−ibi

3. 错排问题 (Derangement)

  • 定义：长度为 nnn 的排列，第 iii 位不能放 iii，求方案数。
  • 递推式：D(n)=(n−1)×(D(n−1)+D(n−2))D(n) = (n-1) \times (D(n-1) + D(n-2))D(n)=(n−1)×(D(n−1)+D(n−2))
    • 分析：考虑数字 nnn 放在了第 kkk 位，那么数字 kkk 可以放在第 nnn 位（转化为 n−2n-2n−2 的子问题），也可以不放在第 nnn 位（转化为 n−1n-1n−1 的子问题）。

4. 卡特兰数 (Catalan Number)

  • 标准公式：Cat(n)=1n+1C2nn=C2nn−C2nn−1Cat(n) = \frac{1}{n+1} C_{2n}^n = C_{2n}^n - C_{2n}^{n-1}Cat(n)=n+11​C2nn​=C2nn​−C2nn−1​
  • 网格路径推广公式（到达 (n,m)(n, m)(n,m) 且不越过对角线的方案）：Cn+mn−Cn+mn+1C_{n+m}^n - C_{n+m}^{n+1}Cn+mn​−Cn+mn+1​
  • 常见应用场景：
    • 长度为 2n2n2n 的合法括号序列数。
    • nnn 个元素的合法出栈序列数。
    • n+2n+2n+2 边凸多边形划分为三角形的方案数。
    • nnn 个节点的二叉树形态数。

五、 矩阵乘法 (Matrix Multiplication)

1. 基础性质

  • 满足结合律：(AB)C=A(BC)(AB)C = A(BC)(AB)C=A(BC)
  • 满足分配律：(A+B)C=AC+BC(A+B)C = AC+BC(A+B)C=AC+BC
  • 注意：不满足交换律！AB≠BAAB \neq BAAB=BA。

2. 核心应用：矩阵快速幂

  • 思想：利用结合律，用 O(log⁡n)O(\log n)O(logn) 的时间计算矩阵的 nnn 次方，常用于加速线性递推式。

六、 博弈论初步 (Game Theory)

1. Nim 游戏 (NIM Game)

  • 规则：若干堆石子，两人轮流取子，每次至少取 1 个，拿光最后石子的人赢。
  • 结论 (Nim 定理)：
    • a1⊕a2⊕⋯⊕an=0a_1 \oplus a_2 \oplus \dots \oplus a_n = 0a1​⊕a2​⊕⋯⊕an​=0 ：必败态 (P-position)。
    • a1⊕a2⊕⋯⊕an≠0a_1 \oplus a_2 \oplus \dots \oplus a_n \neq 0a1​⊕a2​⊕⋯⊕an​=0 ：必胜态 (N-position)。

2. 组合博弈与 SG 函数 (Bash / Sprague-Grundy)

  • 规则扩展：nnn 堆石子，每次只能取 1∼m1 \sim m1∼m 个。
  • 结论：将多堆游戏拆分为独立的单一游戏。求出每一堆的独立状态（如利用余数：a[i] % (m+1)），然后将各堆的胜负状态求异或和。

七、 数论分块 (Integer Partition)

  • 用途：用于快速计算形如 ∑i=1n⌊ni⌋\sum_{i=1}^{n} \lfloor \frac{n}{i} \rfloor∑i=1n​⌊in​⌋ 的式子。
  • 核心思想：⌊ni⌋\lfloor \frac{n}{i} \rfloor⌊in​⌋ 的值会呈现块状分布。分块枚举可以跳过重复的值，时间复杂度从 O(n)O(n)O(n) 降为 O(n)O(\sqrt{n})O(n​)。
  • 结论：对于相同的 ⌊ni⌋\lfloor \frac{n}{i} \rfloor⌊in​⌋，其右边界为 r = n / (n / i)。
