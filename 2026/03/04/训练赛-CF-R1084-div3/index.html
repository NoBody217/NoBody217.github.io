B

题意

  • 给定长为nnn的序列
    • 如果序列非递减那么结束
    • 否则可以删除任意一个元素
  • 求解最终可能剩余的最小元素数

思路

  • 依旧签到
  • 如果最初就是非递减的,那么一个都处理不了
  • 如果最初不是非递减的,那么一定可以删的只剩一个

  • 给定长为nnn的字符串sss,如果满足如下规则就可以si,sjs_i,s_jsi​,sj​变为*
    • si==sj!=∗s_i==s_j!=*si​==sj​!=∗

  • 如果没有可以进行的i,ji,ji,j就结束
  • 判断是否可以将sss中全部元素变为*

思路

  • 典型的括号匹配问题

D

题意

  • 给定长为nnn的排列,序列中有两个传送门
    • 可以从第一个传送门的左侧到第二个传送门的右侧
    • 可以从第二个传送门的右侧到第一个传送门的左侧
  • 请最小化sss的字典序

思路

  • 显然序列被分为内外两段
  • 对于内部,相当于循环转圈,只要找到字典序最小的字符放到开头即可
  • 对于外部,所有比内部大的都应该放到后面

E

题意

  • 给定长为nnn的序列aaa,如果aaa为非递减的则结束
  • Alice 和 Bob 进行博弈,Alice先手
  • 每个玩家在回合中需要选择一个元素xxx,将xxx替换为y,zy,zy,z,满足x=y∗zx=y*zx=y∗z,且1≤y,z≤x1 \le y,z \le x1≤y,z≤x
  • 如果无法进行操作,博弈结束
  • 如果博弈结束时aaa是非递减的,那么Bob获胜,否则Alice获胜

思路

  • 操作其实就是在质因数分解
  • 对于任何一个元素,只要它有超过1个质因子,那么Alice一定获胜
  • 如果所有元素都只有一个质因子,检查一下是否非递减即可

F

题意

  • 贝西有一个反应堆,她有 nnn 个 不同的粒子,每个粒子有两个属性:
    • 能量值 $$x$$:选中它可以产生的能量
    • 反应性 $$y$$:如果选中它,那么整个子集中最多只能再选 $$y$$ 个其他粒子(即子集中粒子总数最多为 $$y+1$$ 个)
  • 她需要选择一个粒子子集,使得子集中的每一个粒子都满足:
    • 子集中的粒子总数 ≤\leq≤ 该粒子的 $$y+1$$
    • 目标是让子集中所有粒子的能量总和最大
  • 现在,有一家商店,里面有 m 个 粒子贝西必须从商店里恰好买一个粒子(可以选择不使用它)
  • 对于商店里的每一个粒子,你需要计算:
    • 如果贝西购买了这个粒子,把它加入自己的粒子库(变成 n+1 个粒子),然后再选择满足约束的最优子集,能产生的最大总能量是多少

思路

  • 排序+乱搞
