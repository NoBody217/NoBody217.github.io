题意

  • 给定一个文本串,多次操作：每次截取指定区间的一段子串,然后拼接到给定的两个目标字符串的中间.
  • 如果两目标字符串不存在或不相邻,则拼接到原文本的末尾.

思路

  • 卡了30分钟最后做完L2回来看才搞出来.
  • 考查 std::string 的组合使用：substr 提取子串,erase 删除原位置子串,find 寻找目标串.
  • 注意0-base下标

L2-048 寻宝图

题意

  • 给定一个二维网格,0 表示水,非 0 表示陆地(其中包含宝藏节点).求总共有多少个独立的岛屿,以及含有宝藏的岛屿数量.

思路

  • 白给bfs
  • 遍历二维矩阵,遇到大于 0 且未被 vis 标记的格子,总岛屿数 +1+1+1,并从该点启动bfs
  • bfs返回岛上有没有宝藏

L2-047 锦标赛

题意

  • 2k2^k2k 名选手打锦标赛,共打 kkk 轮.已知所有选手的战力值(大吃小,相等则随机).给定最终每场比赛的胜负结构,要求还原出一种合法的初始站位排列.

思路

  • 没想出来怎么搞,完全没思路
  • 正解：满二叉树上的 DFS 爆搜.
  • 对于二叉树上的每一场比赛(当前节点),它的胜者要么来自左子树,要么来自右子树.
  • 我们可以写一个递归函数,先假设胜者在左边,验证一下战力值是否合法；如果不行,就回溯(Backtracking),让胜者在右边,交换左右子树的站位继续验证.如果都不行,说明无解.
